# weatherball-site

Public marketing / support / privacy site for the Weatherball iOS app, served via GitHub Pages.

## Pages
- `index.html` — landing page
- `privacy.html` — privacy policy (App Store + RevenueCat requirement)
- `support.html` — support page

## Custom domain
Add a `CNAME` file containing the bare domain (e.g. `weatherball.app`) and point DNS at GitHub Pages once the domain is registered.
